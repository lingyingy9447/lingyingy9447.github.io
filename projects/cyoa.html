Cyoa Project

Context/Content

The prompt of my project is to combine command lines, git, and github to make an interesting choose your own adventure project.

I made a project about escaping prison. There are various different paths the users have to choose to determine their fate. They will either escape or stay in prison till death.

Reflection

One of the most challenging part of creating this project is planning. Planning took us a few days becuase me and my partners didn't have any ideas in mind. Because of the emptiness in my mind, I decided to look up similar topic to get some inspiration.

My biggest takeaway from this project is communication. Communication plays a big part in this projet because we have to notify each other when to pull in the changes we each make. Additionally there might be merge conflict happening because we are editing the same lines, so we have to communicate and split up the work. My next step for this project is to made the md files to html files. I want to make the project more appealing by using html and css.
